# @matthesketh/ink-* packages

Reusable Ink 5 components for modern terminal UIs. Filling the gaps in the Ink ecosystem.

## Core

- `@matthesketh/ink-viewport` — Terminal size hook and viewport-aware layout
- `@matthesketh/ink-scrollable-list` — Windowed scrollable list with follow-cursor
- `@matthesketh/ink-input-dispatcher` — Single-point input routing for Ink apps

## Data Display

- `@matthesketh/ink-table` — Sortable, navigable table with column alignment
- `@matthesketh/ink-chart` — Sparklines, bar charts, and line charts
- `@matthesketh/ink-gauge` — Progress gauge and donut visualisation
- `@matthesketh/ink-tree` — Collapsible tree view for hierarchical data
- `@matthesketh/ink-log-viewer` — Rolling log display with filtering and timestamps
- `@matthesketh/ink-markdown` — Render markdown in the terminal
- `@matthesketh/ink-pager` — Less-like scrollable content viewer

## Input

- `@matthesketh/ink-fuzzy-select` — Filterable select with fuzzy matching
- `@matthesketh/ink-textarea` — Multi-line text input with scrolling
- `@matthesketh/ink-file-picker` — Navigate filesystem and select files

## UI Chrome

- `@matthesketh/ink-status-bar` — Fixed bottom status bar with key hints
- `@matthesketh/ink-modal` — Overlay dialog with border and title
- `@matthesketh/ink-toast` — Timed ephemeral notification messages
- `@matthesketh/ink-keybinding-help` — Auto-generated keybinding reference
- `@matthesketh/ink-task-list` — Step/task progress display with spinners
